class com.haxepunk.Sfx
Sound effect object used to play embedded sounds.
var complete : AudioCompleteCallback
Optional callback function for when the sound finishes playing.
var length(accessor,null) : Float
Length of the sound, in seconds.
var pan(accessor,accessor) : Float
Alter the panning factor (a value from -1 to 1) of the sound during playback.
var playing(accessor,null) : Bool
If the sound is currently playing.
var position(accessor,null) : Float
Position of the currently playing sound, in seconds.
var type(accessor,accessor) : String
Change the sound type. This an arbitrary string you can use to group sounds to mute or pan en masse.
var volume(accessor,accessor) : Float
Alter the volume factor (a value from 0 to 1) of the sound during playback.
function new(source : Dynamic, ?complete : AudioCompleteCallback) : Void
Creates a sound effect from an embedded source. Store a reference to this object so that you can play the sound using play() or loop(). @param source The embedded sound class to use. @param complete Optional callback function for when the sound finishes playing.
function loop(?vol : Float, ?pan : Float) : Void
Plays the sound looping. Will loop continuously until you call stop(), play(), or loop() again. @param vol Volume factor, a value from 0 to 1. @param pan Panning factor, a value from -1 to 1.
function play(?volume : Float, ?pan : Float, ?loop : Bool) : Void
Plays the sound once. @param vol Volume factor, a value from 0 to 1. @param pan Panning factor, a value from -1 to 1. @param loop If the audio should loop infinitely
function resume() : Void
Resumes the sound from the position stop() was called on it.
function stop() : Bool

Stops the sound if it is currently playing.

@return If the sound was stopped.

static function getPan(type : String) : Float

Return the global pan for a type.

@param type The type to get the pan from.

@return The global pan for the type.

static function getVolume(type : String) : Float

Return the global volume for a type.

@param type The type to get the volume from.

@return The global volume for the type.

static function setPan(type : String, pan : Float) : Void

Set the global pan for a type. Sfx instances of this type will add this pan to their own.

@param type The type to set. @param pan The pan value.

static function setVolume(type : String, volume : Float) : Void

Set the global volume for a type. Sfx instances of this type will multiply their volume by this value.

@param type The type to set. @param volume The volume value.
