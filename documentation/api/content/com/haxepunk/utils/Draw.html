class com.haxepunk.utils.Draw
Static class with access to miscellanious drawing functions. These functions are not meant to replace Graphic components for Entities, but rather to help with testing and debugging.
static var blend : flash.display.BlendMode
The blending mode used by Draw functions. This will not apply to Draw.line(), but will apply to Draw.linePlus().
static function circle(x : Int, y : Int, radius : Int, ?color : Int) : Void
Draws a non-filled, pixelated circle. @param x Center x position. @param y Center y position. @param radius Radius of the circle. @param color Color of the circle.
static function circlePlus(x : Int, y : Int, radius : Float, ?color : Int, ?alpha : Float, ?fill : Bool, ?thick : Int) : Void
Draws a circle to the screen. @param x X position of the circle's center. @param y Y position of the circle's center. @param radius Radius of the circle. @param color Color of the circle. @param alpha Alpha of the circle. @param fill If the circle should be filled with the color (true) or just an outline (false). @param thick How thick the outline should be (only applicable when fill = false).
static function curve(x1 : Int, y1 : Int, x2 : Int, y2 : Int, x3 : Int, y3 : Int, ?thick : Float, ?color : Int, ?alpha : Float) : Void
Draws a quadratic curve. @param x1 X start. @param y1 Y start. @param x2 X control point, used to determine the curve. @param y2 Y control point, used to determine the curve. @param x3 X finish. @param y3 Y finish. @param thick The thickness of the curve. @param color Color of the curve @param alpha Alpha transparency.
static function entity(e : com.haxepunk.Entity, ?x : Int, ?y : Int, ?addEntityPosition : Bool) : Void
Draws an Entity object. @param e The Entity to draw. @param x X position. @param y Y position. @param addEntityPosition Adds the Entity's x and y position to the target position.
static function graphic(g : com.haxepunk.Graphic, ?x : Int, ?y : Int) : Void
Draws a graphic object. @param g The Graphic to draw. @param x X position. @param y Y position.
static function hitbox(e : com.haxepunk.Entity, ?outline : Bool, ?color : Int, ?alpha : Float) : Void
Draws the Entity's hitbox. @param e The Entity whose hitbox is to be drawn. @param outline If just the hitbox's outline should be drawn. @param color Color of the hitbox. @param alpha Alpha of the hitbox.

Draws a pixelated, non-antialiased line. @param x1 Starting x position. @param y1 Starting y position. @param x2 Ending x position. @param y2 Ending y position. @param color Color of the line.
static function linePlus(x1 : Int, y1 : Int, x2 : Int, y2 : Int, ?color : Int, ?alpha : Float, ?thick : Float) : Void
Draws a smooth, antialiased line with optional alpha and thickness. @param x1 Starting x position. @param y1 Starting y position. @param x2 Ending x position. @param y2 Ending y position. @param color Color of the line. @param alpha Alpha of the line. @param thick The thickness of the line.

Draws a filled rectangle. @param x X position of the rectangle. @param y Y position of the rectangle. @param width Width of the rectangle. @param height Height of the rectangle. @param color Color of the rectangle. @param alpha Alpha of the rectangle.
static function resetTarget() : Void
Resets the drawing target to the default. The same as calling Draw.setTarget(HXP.buffer, HXP.camera).
static function setTarget(target : flash.display.BitmapData, ?camera : flash.geom.Point, ?blend : flash.display.BlendMode) : Void
Sets the drawing target for Draw functions. @param target The buffer to draw to. @param camera The camera offset (use null for none). @param blend The blend mode to use.
