class com.haxepunk.utils.Data
Static helper class used for saving and loading data from stored cookies.
static var id : String
If you want to share data between different SWFs on the same host, use this id.
static function load(?file : String) : Void
Overwrites the current data with the file. @param file The filename to load.
static function read(name : String, ?defaultValue : Dynamic) : Dynamic
Reads a property from the data object. @param name Property to read. @param defaultValue Default value. @return The property value, or defaultValue if the property is not assigned.
static function readBool(name : String, ?defaultValue : Bool) : Bool
Reads a Boolean from the current data. @param name Property to read. @param defaultValue Default value. @return The property value, or defaultValue if the property is not assigned.
static function readInt(name : String, ?defaultValue : Int) : Int
Reads an int from the current data. @param name Property to read. @param defaultValue Default value. @return The property value, or defaultValue if the property is not assigned.
static function readString(name : String, ?defaultValue : String) : String
Reads a String from the current data. @param name Property to read. @param defaultValue Default value. @return The property value, or defaultValue if the property is not assigned.
static function save(?file : String, ?overwrite : Bool) : Void
Overwrites the file with the current data. The current data will not be saved until this function is called. @param file The filename to save. @param overwrite Clear the file before saving.
static function write(name : String, value : Dynamic) : Void
Writes a Dynamic object to the current data. @param name Property to write. @param value Value to write.
