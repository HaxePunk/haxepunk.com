class com.haxepunk.graphics.Canvas
extends com.haxepunk.Graphic
A multi-purpose drawing canvas, can be sized beyond the normal Flash BitmapData limits.
var alpha(accessor,accessor) : Float
Change the opacity of the Canvas, a value from 0 to 1.
var angle : Float
Rotation of the canvas, in degrees.
var blend : flash.display.BlendMode
Optional blend mode to use (see flash.display.BlendMode for blending modes).
var color(accessor,accessor) : Int
The tinted color of the Canvas. Use 0xFFFFFF to draw the it normally.
var height(accessor,null) : Int
Height of the canvas.
var scale : Float
Scale of the canvas, effects both x and y scale.
var scaleX : Float
X scale of the canvas.
var scaleY : Float
Y scale of the canvas.
var width(accessor,null) : Int
Width of the canvas.
function new(width : Int, height : Int) : Void
Constructor. @param width Width of the canvas. @param height Height of the canvas.
function draw(x : Int, y : Int, source : flash.display.BitmapData, ?rect : flash.geom.Rectangle) : Void
Draws to the canvas. @param x X position to draw. @param y Y position to draw. @param source Source BitmapData. @param rect Optional area of the source image to draw from. If null, the entire BitmapData will be drawn.
function drawGraphic(x : Int, y : Int, source : com.haxepunk.Graphic) : Void
Draws the Graphic object to the canvas. @param x X position to draw. @param y Y position to draw. @param source Graphic to draw.
function drawRect(rect : flash.geom.Rectangle, ?color : Int, ?alpha : Float) : Void
Draws over a rectangular area of the canvas. @param rect Drawing rectangle. @param color Draw color. @param alpha Draw alpha. If < 1, this rectangle will blend with existing contents of the canvas.
function fill(rect : flash.geom.Rectangle, ?color : Int, ?alpha : Float) : Void
Fills the rectangular area of the canvas. The previous contents of that area are completely removed. @param rect Fill rectangle. @param color Fill color. @param alpha Fill alpha.
function fillTexture(rect : flash.geom.Rectangle, texture : flash.display.BitmapData) : Void
Fills the rectangle area of the canvas with the texture. @param rect Fill rectangle. @param texture Fill texture.
function shift(?x : Int, ?y : Int) : Void
Shifts the canvas' pixels by the offset. @param x Horizontal shift. @param y Vertical shift.
