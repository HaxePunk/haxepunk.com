class com.haxepunk.graphics.Animation
Template used by Spritemap to define animations. Don't create these yourself, instead you can fetch them with Spritemap's add().
var frameCount(default,null) : Int
Amount of frames in the animation.
var frameRate(default,null) : Float
Animation speed.
var frames(default,null) : Array<Int>
Array of frame indices to animate.
var loop(default,null) : Bool
If the animation loops.
var name(default,null) : String
Name of the animation.
var parent(null,accessor) : Spritemap
function new(name : String, frames : Array<Int>, ?frameRate : Float, ?loop : Bool) : Void
Constructor. @param name Animation name. @param frames Array of frame indices to animate. @param frameRate Animation speed. @param loop If the animation should loop.
function play(?reset : Bool) : Void
Plays the animation. @param reset If the animation should force-restart if it is already playing.
