package com.haxepunk.masks

class Imagemask


extends Pixelmask

A bitmap mask used for pixel-perfect collision.

Example usage:

class Object extends Entity
{
    public function new()
    {
        super();
        var sprite = new Image("myimage.png", 100, 100);
        graphic = sprite;
        mask = new Imagemask(sprite);
    }
}

Remember to call "mask.update()" when you update the image.

If you are using RenderMode.HARDWARE mode, Imagemask will still work, but only if your Image source is created with a BitmapData. AtlasData is not currently supported.
